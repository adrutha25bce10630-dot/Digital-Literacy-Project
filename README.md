# Digital Literacy Project

## Project Overview

This repository contains my Digital Literacy Project completed as part of the CSE0001 course.

In this project, I assumed the role of a **Student Digital Ambassador**, where the objective was to help peers understand and effectively use digital tools, platforms, and communication methods in an academic environment.

The project focuses on developing essential digital skills such as:

* Creating digital content
* Building a professional online presence
* Practicing coding and collaboration tools
* Writing professional emails
* Understanding cybercrime and online safety
  
### Task 1: Digital Literacy Infographic

A digital infographic was created using Canva to explain key aspects of digital literacy, including safe internet practices, use of digital tools, and email etiquette.

The infographic was designed to be clear and visually structured, with content divided into sections for better readability. One of the main challenges faced was balancing the amount of information while maintaining a clean layout.

This task helped improve my ability to present information visually and strengthened my understanding of digital literacy concepts.

### Task 2: Student Digital Portfolio

Profiles were created and developed on the following platforms:

* GitHub
* LinkedIn
* Kaggle

Each platform serves a specific purpose:

* GitHub is used for sharing code and managing projects
* LinkedIn is used for professional networking and showcasing achievements
* Kaggle is used for practicing data science and working with datasets

I plan to actively use these platforms over the next four years to build a strong portfolio, improve technical skills, and prepare for future career opportunities.

### Task 3: Coding and Collaboration Platforms

For coding practice, I used HackerRank and completed beginner-level problems to improve my logical thinking and programming skills.

For collaboration, I created a Digital Literacy Awareness Quiz using Google Forms. The form included multiple types of questions and was designed to be simple and user-friendly.

This task helped me understand how to use online platforms for both skill development and data collection, which will be useful for academic projects and research.

### Task 4: Professional Email and Etiquette

This task focused on understanding proper digital communication. Professional email drafts were created, following correct structure, tone, and formatting.

A scenario was analyzed where poor communication in a group project led to confusion due to unclear responsibilities. This highlighted the importance of clear, detailed, and timely communication.

Additionally, a social media etiquette checklist was created to promote responsible online behavior.

### Task 5: Cybercrime Awareness

This task involved studying cybercrime, particularly phishing and online fraud. It highlighted how easily users can be misled by realistic-looking messages and scams.

A case study and prevention checklist were created to spread awareness about safe online practices.

As a personal takeaway, I learned the importance of verifying sources before clicking links or sharing information, which will help protect against cyber threats.
